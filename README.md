# SkyPulse

SkyPulse is a weather monitoring and forecasting application built with React and Vite. It allows users to view current weather conditions, as well as a 6-day weather forecast. The app provides details such as temperature, wind speed, humidity, and weather conditions (e.g., cloudy, foggy, clear, thunderstorm, rain, etc.) all sourced from a weather API.
In this Application you can search City name and get all data . if city not found then it shows city not found.

# Visit https://weather-application-six-alpha.vercel.app/

## Features

- **Current Weather:** Displays real-time weather information including temperature, wind speed, and humidity.
- **Weather Conditions:** Shows the current weather condition (e.g., cloudy, foggy, rain, clear, thunderstorm).
- **6-Day Forecast:** Provides a weather forecast for the next 6 days, including temperature and conditions.
- **Responsive Design:** Works seamlessly on desktop and mobile devices.
- **API Integration:** Fetches weather data from a reliable weather API.
- **Search:** In this you can search your own city.

## Technologies Used

- **React** - A JavaScript library for building user interfaces.
- **Vite** - A modern build tool that provides fast performance and optimized bundling.
- **Weather API** - Data source for current weather and forecasts.
